    Why not direct imports?

    Apple Music, Tidal, and Amazon Music require server-signed tokens or commercial partner access that a static site can't safely ship. YouTube/Google OAuth is possible but adds another setup step. So instead, this tool hands you per-track deep links — click each to add it on the target service, or export the list for tools like Soundiiz / TuneMyMusic / Spotlistr if you want batch import.

    Nothing you paste leaves your browser except the calls to Spotify's own API and the search links you click.
